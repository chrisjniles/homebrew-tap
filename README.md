# homebrew-tap

A [Homebrew](https://brew.sh) tap for my personal tools.

## Install

```sh
brew install chrisjniles/tap/mcmobsheet
```

Or tap first, then install:

```sh
brew tap chrisjniles/tap
brew install mcmobsheet
```

## Formulae

- **[mcmobsheet](https://github.com/chrisjniles/mcmobsheet)** — translate Minecraft
  `/summon` and `/setblock` data into readable info.
